APL Execute Commands Element

Prepares an APL Command Execute Directive from other APL Command Elements.

Currently Supported Commands are:

  • Auto Page
  • Backstack Clear
  • Backstack Go Back
  • Clear Focus
  • Finish
  • Idle
  • Reinflate
  • Scroll
  • Scroll to Component
  • Scroll to Index
  • Send Event
  • Set Focus
  • Set Value
  • Speak Item
  • Speak List

When to use?

When you use some APL Commands.

How to use:

  1. If you are planing to execute commands on an another Document, please specify the token of that Document, otherwise the commands will be executed on the current APL Document..
  2. Place APL Commands and configure them
