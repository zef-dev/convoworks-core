Standard Card Element

A Standard card displays plain text, but can include an image.

This card can display on an Alexa-enabled device with a screen, such as an Echo Show. In addition, it can be displayed in the Amazon Alexa App, the companion app available for Fire OS, Android, iOS, and desktop web browsers.

To view the card in the Alexa App, open the Alexa app and navigate to the Activity page.

Fields to be configured:

  • Title - Title of the card.
  • Text - Content of the card. Use either "\r\n" or "\n" within the text to insert line breaks.
  • Small Image URL - The URL of a small version of the image to display.
  • Large Image URL - The URL of a large version of the image to display.

For more details, please visit the Alexa Developer Documentation at Create a card to display text and an image
