Dialog Delegate Element

Delegates the slot filling conversation to Alexa Dialog.

Parameters:

  • Action - One of the following actions: "Delegate", "Delegate and Update Incoming Intent" or "Delegate and Update Another Intent".
  • Intent - Is available when Delegate and Update Another Intent is selected in the Action field. You should update another intent only when the dialog state is "COMPLETED".
  • Intent Slot Values - Is available when Delegate and Update Incoming Intent or Delegate and Update Another Intent is selected in the Action field. This field defines which slot names with what values will be filled in "COMPLETED" or "IN_PROGRESS" dialog states.

For more details, please visit the Alexa Developer Documentation at Delegate directive
