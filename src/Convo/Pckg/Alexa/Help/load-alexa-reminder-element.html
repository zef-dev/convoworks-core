Load Alexa Reminder Element

Returns single reminder data by an alert token id.

Parameters:

  • Reminder ID - ID of the existing and ACTIVE reminder of an Alexa Skill
  • Return Variable Name - Default status, name of the variable that contains additional information (appointment, timezone)
