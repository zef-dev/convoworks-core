Get Amazon Customer Profile Element

Retrieves data from default active account that is on Alexa Device with granted user permissions.

When to use?

When you want to retrieve customer information from the Amazon User Account and use it in your Alexa Skill.

How to use:

  1. Make sure you already have configured and propagated your service to Amazon Alexa with checked permissions for your needs in Amazon Alexa Skill Permissions:
    • Full Name or Given Name
    • Customer Email Address
    • Customer Phone Number
  2. In case the customer profile could be fetched successfully, OK flow will be executed.

    After the user has enabled the skill and granted permissions, you'll be able to access the customer profile data like in the following examples:

    • ${customer_profile_status.fullName} when Full Name permission was granted by the user
    • ${customer_profile_status.givenName} when Given Name permission was granted by the user
    • ${customer_profile_status.emailAddress} when Given Name permission was granted by the user
    • ${customer_profile_status.phoneNumber.countryCode} when Phone Number permission was granted by the user
    • ${customer_profile_status.phoneNumber.phoneNumber} when Phone Number permission was granted by the user

    In case the customer profile could not be fetched successfully, On Permission Not Granted or NOK flow will be executed

    You can access the error message like this ${customer_profile_status.error}.
