Start Radio Stream Playback

Starts radio stream playback by the provided parameters. It also closes the current session - radio players are device feature and are not working in standard service session.

Parameters:

  • Stream URL - Secure URL to the stream of the radio station which can be in various formats.
  • Radio Station Name - Name of the radio station. (optional)
  • Radio Station Slogan - Slogan of the radio station, something like: "Croatia's only Dance Radio." (optional)
  • Radio Station Logo URL - Secure URL to the logo of the radio station. (optional)

Just fill out the parameters mentioned below, and your Alexa Device will be playing the radio stream. At lease the Stream URL should be filled out, and you are good to go.
