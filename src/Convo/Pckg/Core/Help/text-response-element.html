Text response

Text which will be responded to the user. Response can be default response or reprompt.

When to use?

When we want to create a response(speech) to the user via device.

How to use:

  1. Select the TYPE from the drop-down
  2. Select the ALEXA DOMAIN from the drop-down
  3. Select the ALEXA EMOTION from the drop-down
  4. Select the ALEXA EMOTION INTENSITY from the drop-down
  5. Add message to TEXT
  6. Select APPEND and text will be appended to the preceding sentence (if any) instead of creating a new one.

Example of usage for creating simple text response

We're going to create simple text response

  • TYPE:

    Default

  • ALEXA DOMAIN:

    Normal

  • ALEXA EMOTION:

    Neutral

  • ALEXA EMOTION INTENSITY:

    Medium

  • TEXT:

    Hello world!
