Element randomizer

Picks just one child element end executes it.

When to use?

When we want to randomize our output.

How to use:

  1. Select the Wild(picks random output every time and executes it) or Smart(picks different random output every time and executes it) from ELEMENT STORAGE MODE drop-down
  2. Add random string to NAMESPACE when using Smart mode
  3. Select the SCOPE TYPE from the drop-down
  4. Scope type installation is for the duration of the whole installation and has the longest life span
  5. Scope type session is for the duration of the session
