NOP filter

No operation - does nothing.

When to use?

When we want to use this filter to test workflows.

How to use:

  1. In Simple Processor in Request filters container add NOP filter
  2. In the IS EMPTY drop-down select empty or match
  3. Empty - Will not match, this setting will not activate the processor's OK flow.
  4. Match - Will accept the incoming value and trigger the processor's OK flow.
