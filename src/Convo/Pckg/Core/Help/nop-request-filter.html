NOP filter

No operation - does nothing.

When to use?

When we want to use this filter to test workflows.

How to use:

  1. Place this element in the Request Filters container on a Simple Processor
  2. In the Is Empty drop-down, select empty or match
    • Empty - Will not match, this setting will not activate the processor's OK flow.
    • Match - Will accept the incoming value and trigger the processor's OK flow.
