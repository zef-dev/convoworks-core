Seek Audio Playback by Search

Initiates audio playback by search in the current playlist. It also closes the current session - audio players are device feature and are not working in standard service session.

You can pass the Search Term of the desired song to play. If the Search Term has not matches, it will execute fallback flow.
If you are not sure will there be any songs to play, use this element with the Media info element which contains separate flows for cases with and without data.
