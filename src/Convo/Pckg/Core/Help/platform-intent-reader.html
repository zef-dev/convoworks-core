Platform Intent

Reads platform intents that have not been defined in the Convoworks editor. You can manually enter platform intents to be checked for in incoming requests.

When to use?

If there is an intent present on only one or a couple of vendors, but nowhere else (e.g. a specific Amazon built-in intent), you can use this filter to specifically deal with those cases.

How to use

  1. Write the name of the intent in the Intent field. This is case sensitive.
  2. You can add Predefined values if you want to make them available on the result object.
  3. You can rename incoming slot values if you want to, by setting up Rename values pairs.
