Card

Displays simple card layout, with title, some text and one image on the right. Works for Alexa and Assistant. Use just single card per step. Ignored on non display capable devices.

Example usage

Card field value filling example:

  • Title:

    ${data.propertyBestSuitedForTitle}

  • Subtitle:

    ${data.propertyBestSuitedForSubtitle}

  • Description 1:

    ${data.somePrimaryDescription}

  • Image url:

    ${data.someImageLink}

  • Image text:

    ${data.someImageText}
