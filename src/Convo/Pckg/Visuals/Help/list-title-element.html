List Title Element

Initiatilizes a List view and sets the title line. Works on Alexa and Google Assistant. Use just a single card per step. Ignored on non display capable devices.

For adding items to the list, use the List Item element.

When to use?

Whenever you want to display a graphical list. An example might be a list of products, or maybe a list of ingredients required for a recipe.

How to use

  1. Place this element and give the list a List title.
  2. Pick between List and Carousel layouts from the List template menu.
    • List is a vertical, top-to-bottom layout.
    • Carousel is a horizontal layout, more suited to something like an image gallery.
  3. Place List Items after this element.
